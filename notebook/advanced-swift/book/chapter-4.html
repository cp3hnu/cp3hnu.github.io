# a ?? b ?? c 和 (a ?? b) ?? c 的区别

对于一层可选值他们是一样的，但是对于双层嵌套可选值就不一样了

let s1: String?? = nil // nil
(s1 ?? "inner") ?? "outer" // inner
let s2: String?? = .some(nil) // Optional(nil) 
(s2 ?? "inner") ?? "outer" // outer
